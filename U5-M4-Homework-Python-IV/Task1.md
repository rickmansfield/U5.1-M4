# Unit 5.1 M4 Task 1
What are the two primary weaknesses of a static array as a data structure?
A) 
1. O(n) lookups
2. O(n) deletions

B) 
1. O(n) appends
2. Slow lookups

C) 
1. Fixed Size
2. O(n) insertions and deletions
